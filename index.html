📊 Sincronización con Google Sheets
Tiempo real · Sin conflictos · Gratis · Sin límites
1
Abre Google Sheets
Crea una hoja nueva en sheets.google.com. Ponle el nombre que quieras.
2
Abre el editor de Apps Script
Dentro de Sheets → menú Extensiones → Apps Script. Borra el código que hay y pega el script que te hemos dado.
3
Implementa como Web App
Clic en Implementar → Nueva implementación → Tipo: Aplicación web → Acceso: Cualquier usuario → Implementar. Copia la URL.
4
Pega la URL del Web App aquí
🔒 La URL se guarda en este dispositivo.
Todos los dispositivos deben apuntar al mismo Google Sheet.
Tip: Para no configurar cada dispositivo, pide al admin que
hardcodee la URL en el código fuente (constante HARDCODED_SHEETS_URL).
